# Marsdiep_data
This repository contains the code used in the paper Marine carbonate system variability from tidal to seasonal timescales at the interface between the North Sea and Wadden Sea (https://egusphere.copernicus.org/preprints/2025/egusphere-2025-5050/). It includes all scripts used to perform the calculations presented in the study, and generate the figures and plots shown in the paper.
